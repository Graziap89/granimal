# Granimal's Frontend
## Introduction
The application frontend allows the user to interact with the system. The user through the interface can search for series and obtain the main information. 


#### Installation and launch

```bash

cd frontend

npm i

npm run dev
```
#### Technologies used
Our frontend uses

- **Typescript** 
- **React** 
- **Tailwind CSS**
- **DaisyUI**
- **Vite**
